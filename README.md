# Code with Samiha

A free, beginner-friendly web coding platform built specifically for girls who want to learn programming at their own pace.

## Project Description

Code with Samiha is a frontend web application built for CSCI390 Web Programming — Phase 2. It provides structured coding tracks, interactive lessons, a learning roadmap, and a community page. The interface is fully responsive and works on both desktop and mobile.

**Pages included:**
- **Home** (`index.html`) — hero section, language tracks, how it works, testimonials, CTA
- **Languages** (`pages/languages.html`) — browsable, filterable list of all language tracks
- **Lessons** (`pages/lessons.html`) — lesson sidebar, code viewer, live try-it editor
- **Roadmap** (`pages/roadmap.html`) — visual timeline of the learning path
- **Community** (`pages/community.html`) — forum-style posts with tag filtering

## Technologies Used

- HTML5 & CSS3
- Bootstrap 5.3
- Vanilla JavaScript (ES5/ES6)
- Google Fonts (Fraunces, Plus Jakarta Sans, JetBrains Mono)
- Git & GitHub for version control

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/code-with-samiha.git
   ```
2. Open `index.html` in any modern browser — no build step required.
3. All pages link to each other relatively, so the folder structure must stay intact.
